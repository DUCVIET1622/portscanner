## Tính năng

| Tính năng | Mô tả |
|-----------|-------|
| ✅ **TCP Connect Scan** | Phát hiện cổng mở bằng kết nối TCP đầy đủ |
| ✅ **Hostname Resolution** | Tự động phân giải domain → IP |
| ✅ **Tùy chỉnh dải cổng** | `1-1000`, `80`, `22-443`, `1-65535` |
| ✅ **Service Detection** | Nhận diện tên dịch vụ từ cổng |
| ✅ **Progress Indicator** | Hiển thị tiến trình mỗi 100 cổng |
| ✅ **Error Handling** | Chống crash — bắt mọi exception |

| ✅ **Không dependencies** | Chỉ dùng thư viện chuẩn Python |
